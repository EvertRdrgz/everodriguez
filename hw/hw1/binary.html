Uses

The use of binary numbers in computer hardware is due to the fact that it is easy to represent relative high or low volatages in computer hardware using this base 2 system.

Theory

A binary number is only made up of two symbols, 0 and 1, called bits. Just like each digit in a standard decicaml number represents 10^n, each number in a binary system represents 2^n.

Example

If we want to conver the decimal number to 25 to binary, we need 5 bits. The left-most number has the value of 16, because 2^4 = 16. The number to the right of it has value of 8, because 2^3 = 8 and so on. Thus, the number 25 in decimal can be written as 11001.
