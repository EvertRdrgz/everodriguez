Uses

Computer hardware system need to have system for adding binary numbers, such as the one in the previous page. However, we also need a sytem for representing negative numbers. For this we have 1s and 2s complements.

Theory

Using 1s and 2s complement allows us to do arithmitic using binary numbers. These systems allows us to represent both negative and postive numbers in one system, just like the one we are used to with decicaml numbers.

Example

To get the first complement of a binary number, all we have to do is flip the digits, all the 1s becoome 0s and 0s become 1s. In our previous exmaple, 25 = 11001 in binary. In 1s complement, it would equal to 00110. To get the 2s complement of a number, we simply add 1 to a 1s complement number. So our number is now 00111 in 2s complement.
